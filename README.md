# tuiper
A terminal user-interface for practicing typing and for racing against fellow typists.

## Features
- Solo practicing mode
- Real-time multiplayer between two players (locally)

## Current installation steps
  1. Install Rust
  2. ``git clone https://github.com/GopeshSingal/tuiper.git``
  3. ``export WS_URL=ws://35.223.202.42:8080/ws``
  4. ``cargo run --release -p tuiper``
